    MySQL忘记密码进行修改

    如果MySQL密码忘记了,一共有步骤:

    0:进入CMD,cd 首先进入mysql.exe 也就是 C:\Program Files\MySQL\MySQL Server 5.7\bin 这个目录

    1:Ctrl+R打开二个CMD

    2:找到Mysql.exe,一般存储在C盘 中的MySQL\bin目录下

    3:在第一个CMD中输入mysqld --skip-grant-tables; 回车 跳过用户验证

    4:打开第二个CMD,输入MySQL,回车,再输入show databases;

    5:选择数据库 use mysql;

    6:update mysql.user set authentication_string=password('root') where user='root';
